CSS

CSS staat voor Cascading Style Sheets. Het is een programmeertaal die zorgt voor de opmaak van HTML bestanden. Het zorgt er ook voor dat je website er cooler uitziet als je daarnaast ook HTML gebruikt. CSS zorgt dus voor de opmaak, maar om elementen verschillend op te maken moet er natuurlijk wel een manier zijn om je op specifieke elementen te richten. Je kunt dit doen door de HTML codes die gebruikt worden. Het voordeel van CSS is dat CSS style sheets worden door alle webbrowsers uitstekend ondersteund. Misschien inmiddels zelfs al beter dan de ouderwetse methode van opmaak in de HTML-code zelf.

HTML

HTML staat voor Hyper Text Markup Language en is een programmeertaal die wordt gebruikt voor het schrijven van webpagina's. Een HTML pagina begint altijd met een DOCTYPE declaratie. Net zoals elke andere vakman heeft ook een front-end ontwikkelaar een uitgebreide set aan tools ter beschikking. Gaande van basis functionaliteiten als Color Coding of IntelliSense, tot de implementatie van geavanceerde bibliotheken. Het leren van HTML, CSS en andere programmeertalen ziet er op het begin vaak moeilijk uit maar, wanneer je beetje bij beetje meer leert van deze talen merk je vanzelf dat het zo moeilijk nog niet is.
